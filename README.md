## インストール方法

1. [Releases](https://github.com/kract/colorpicker/releases)にアクセスする。
2. 「kpicker.zip」を選択しダウンロードし、解凍します。
3. [chrome://extensions/](chrome://extensions/)にアクセスする。
4. 「デベロッパーモード」を有効にする。
5. 「パッケージ化されていない拡張機能を読み込む」から解凍したフォルダを選択する。
